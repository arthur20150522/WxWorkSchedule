# 执行远程部署

我将直接在你的本地终端运行刚才创建的 `deploy.ps1` 脚本，这将自动完成以下步骤：

1.  **本地构建**：编译前端 (`client`) 和后端 (`server`) 代码。
2.  **文件上传**：使用 `scp` 将构建好的文件传输到服务器 `43.153.88.215` 的 `/www/wwwroot/WxWork` 目录。
3.  **远程安装与启动**：
    *   通过 `ssh` 在服务器上安装生产环境依赖。
    *   使用 `pm2` 启动或重载后端服务。

**注意**：
*   此过程依赖你本地已配置好服务器的 SSH 免密登录（或已保存 SSH Key）。如果脚本在连接时卡住等待密码，可能需要你手动输入密码或配置免密。
*   Nginx 配置需要根据服务器实际情况手动调整（脚本不会自动修改 Nginx 配置以防止覆盖现有设置）。
